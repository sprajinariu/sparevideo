# ViBe Phase 2 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Land the RTL ViBe background-subtraction block alongside the existing EMA path, gated by `CFG.bg_model`, with parametric K ∈ {8, 20}, an external-init `$readmemh` ROM path for ghost-free demos, and the `cfg_t` field-list trim from the contract spec.

**Architecture:** New modules `axis_motion_detect_vibe` (AXIS wrapper) + `motion_core_vibe` (algorithm core) mirror the existing EMA pair's split. A compile-time `generate-if` at `sparevideo_top.sv` selects EMA or ViBe per `CFG.bg_model`. The ViBe block uses a single dual-port BRAM with byte-enables for the K-sample bank (master design §6), Xorshift32 PRNG (master design §7), and a 4-deep defer-FIFO for rare joint-fire collisions. K=20 frame-0 init uses three parallel Xorshift32 streams with seed-XOR offsets (B1 from design §3.3) to avoid serial-correlation artifacts. External init via `$readmemh`-loaded `.mem` file generated by a new Python helper from the input source's first N frames (lookahead median).

**Tech Stack:**
- **RTL:** SystemVerilog, Verilator. New: `hw/ip/motion/rtl/{axis_motion_detect_vibe.sv, motion_core_vibe.sv}`.
- **Verification:** Verilator unit TB at `hw/ip/motion/tb/tb_axis_motion_detect_vibe.sv` covering parametric K, both init paths, backpressure, and elaboration-time misconfig errors. Top-level integration via existing `make run-pipeline` at TOLERANCE=0 vs. Python ref.
- **Python:** `py/gen_vibe_init_rom.py` (new), `py/tests/test_vibe_init_rom.py` (new), edits to `py/profiles.py`, `py/models/motion_vibe.py`, `py/tests/test_profiles.py`, `py/tests/test_motion_vibe.py`.
- **Build:** `make lint`, `make test-ip` (gains `test-ip-vibe`), `make test-py`, `make run-pipeline`, `make sw-dry-run`.

**Reference:** [Phase 2 design spec](2026-05-06-vibe-phase-2-design.md) — read all sections before starting. [Contract spec](2026-05-06-vibe-rtl-cfg-contract-design.md) — drives Phase A. [Master design](2026-05-01-vibe-motion-design.md) §3/§6/§7 — algorithm, BRAM layout, PRNG.

```python
"""ViBe sample-bank ROM generator.

Reads input.bin, computes per-pixel lookahead-median over first N frames,
applies scheme-c noise jitter via Xorshift32 (single stream at K=8, three
parallel streams at K=20), writes a $readmemh-format hex file matching the
RTL BRAM layout.
"""
import argparse, struct, sys
import numpy as np


def xorshift32(s):
    s = (s ^ (s << 13)) & 0xFFFFFFFF
    s = (s ^ (s >> 17)) & 0xFFFFFFFF
    s = (s ^ (s << 5))  & 0xFFFFFFFF
    return s


def luma(rgb):
    """RGB → Y (BT.601, integer rounded). rgb shape (..., 3) uint8."""
    r, g, b = rgb[..., 0].astype(np.int32), rgb[..., 1].astype(np.int32), rgb[..., 2].astype(np.int32)
    return ((299 * r + 587 * g + 114 * b + 500) // 1000).clip(0, 255).astype(np.uint8)


def compute_bank(input_path, width, height, k, lookahead_n, seed):
    # Read frames from input.bin (12-byte header + raw RGB bytes)
    with open(input_path, "rb") as f:
        hdr = f.read(12)
        w, h, total_frames = struct.unpack("<III", hdr)
        assert w == width and h == height, f"input dims {w}x{h} != {width}x{height}"
        # Decide N
        n = total_frames if lookahead_n == 0 else min(lookahead_n, total_frames)
        rgb = np.frombuffer(f.read(n * h * w * 3), dtype=np.uint8).reshape(n, h, w, 3)

    # Per-pixel temporal median
    y = luma(rgb)  # (n, h, w)
    median = np.median(y, axis=0).astype(np.uint8)  # (h, w)

    # Generate K noise-jittered slots per pixel using Xorshift32 streams
    SEED_DOMAIN_OFFSET = 0x4F495E11  # avoid collision with self-init PRNG
    bank = np.zeros((h, w, k), dtype=np.uint8)
    state0 = (seed ^ SEED_DOMAIN_OFFSET) & 0xFFFFFFFF
    state1 = (state0 ^ 0x9E3779B9) & 0xFFFFFFFF
    state2 = (state0 ^ 0xD1B54A32) & 0xFFFFFFFF

    for y_idx in range(h):
        for x_idx in range(w):
            # Advance streams (mirror RTL: one advance per pixel for all active streams)
            state0 = xorshift32(state0)
            if k == 20:
                state1 = xorshift32(state1)
                state2 = xorshift32(state2)
                pool = (state2 << 64) | (state1 << 32) | state0
            else:
                pool = state0
            # Slice K × 4-bit nibbles → noise [-8, +7]
            for slot in range(k):
                nibble = (pool >> (slot * 4)) & 0xF
                noise = nibble - 8
                val = int(median[y_idx, x_idx]) + noise
                bank[y_idx, x_idx, slot] = max(0, min(255, val))
    return bank


def write_mem(out_path, bank, width, height, k, seed, lookahead_n):
    with open(out_path, "w") as f:
        f.write(f"// generated by py/gen_vibe_init_rom.py\n")
        f.write(f"// width={width} height={height} K={k} seed=0x{seed:08X} lookahead_n={lookahead_n}\n")
        for y in range(height):
            for x in range(width):
                # MSB-first packing: {slot[k-1], ..., slot[0]}
                hex_str = "".join(f"{bank[y, x, k - 1 - slot]:02x}" for slot in range(k))
                f.write(hex_str + "\n")


def main():
    ap = argparse.ArgumentParser()
    ap.add_argument("--input",  required=True)
    ap.add_argument("--output", required=True)
    ap.add_argument("--width",  type=int, required=True)
    ap.add_argument("--height", type=int, required=True)
    ap.add_argument("--k",      type=int, choices=[8, 20], required=True)
    ap.add_argument("--lookahead-n", type=int, required=True)
    ap.add_argument("--seed",   type=lambda s: int(s, 0), required=True)
    args = ap.parse_args()
```
